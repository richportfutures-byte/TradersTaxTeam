# Schedule C Mapping Notes

## Purpose
Map common business-use operating expenses tracked in the app to rough planning categories that may later inform Schedule C review.

## Example mappings
- platform_software -> software/subscriptions
- market_data -> subscriptions/data
- analytics_research -> research tools
- internet_phone -> allocated communications expense
- cloud_it_security -> IT services
- professional_fees -> CPA/legal/professional
- office_supplies -> supplies
- home_office -> separate review required
- bank_fees -> banking/processing fees
- education -> conservative review required; not always deductible

## Caution
These mappings are planning aids only. Final categorization belongs in a human tax review workflow.
