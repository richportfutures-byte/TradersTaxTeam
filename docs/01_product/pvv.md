# Product Vision and Values (PVV)

## Vision
Create a disciplined personal finance operating system for an active futures trader that turns broker exports into trustworthy operating reality, not vanity P&L.

## Core values

### 1. Truth over convenience
If the numbers do not reconcile, the product must say so plainly.

### 2. Narrow and useful
The app should solve one user’s monthly workflow well before expanding into broad accounting or filing ambitions.

### 3. Conservative planning posture
Tax logic should bias toward caution and explicit assumptions, not maximum optimization theater.

### 4. User-owned control
The user should be able to run, inspect, and export their data locally.

### 5. Durable records
Exports and audit notes should survive beyond any single UI session.

## Value proposition
LedgerFlow helps an active trader understand:
- real profitability after both execution costs and business overhead
- whether broker-imported values can be trusted
- what cash should be reserved for taxes
- what structured record set exists for later filing support
